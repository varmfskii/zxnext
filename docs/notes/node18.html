Timex Hi-Colour Mode

Timex mode 2

This mode is a #tex2html_wrap_inline180# video mode with #tex2html_wrap_inline182# colour attribute mapping on a #tex2html_wrap_inline184# grid. It is selected by writing 2 to the Timex ULA port ($ff). Pixel mapping in this mode is the same as in ZX Spectrum mode using the bitmap 1 area based on #math73##tex2html_wrap_inline186#. The colour attributes use the bitmap 2 area with #tex2html_wrap_inline188# colour attribute areas corresponding to the addresses #math74##tex2html_wrap_inline190#.
