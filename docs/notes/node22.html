ULA Clipping

All of the modes in the ULA group share a single clipping register, nextreg $1A (Clip Window ULA). The clip index may alternately be set using nextreg $1C. This is expecially useful for reading the current clipping coordinates as reads on the clipping register do not change the index. Note that clipping coordinates are based on a full display area for the mode of even though not all ULA group modes have that resolution.

(R/W) $1A (26) Clip Window ULA (and LoRes )

bits 7-0 = Clip window coordinate (inclusive)
  • 1st write = X1 position (soft reset = 0)
  • 2nd write = X2 position (soft reset = 255)
  • 3rd write = Y1 position (soft reset = 0)
  • 4rd write = Y2 position (soft reset = 191)
 Reads do not advance the clip position
  LoRes may get a separate clip window in the future
