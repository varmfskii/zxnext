ULA Scrolling

The ULA group has two sets of scrolling registers. One for the the legacy modes (ZX Spectrum, Alternate Page, Timex Hi-Resoulution, and Timex Hi-colour) and a second set for the two ZX Spextrum Next specific LoRes modes. The Registers for the legacy modes are nextreg $26 (ULA X Scroll) and $27 (ULA Y Scroll). Timex Hi-resolution scrolls as if it was a pixel screen mode.

(R/W) $26 (38) ULA X Scroll

  • bits 7-0 = X Offset (0-255) (soft reset = 0)

(R/W) $27 (39) ULA Y Scroll

  • bits 7-0 = Y Offset (0-191) (soft reset = 0)
