Layer 2 Switching

Layer 2 switching can allow any 16k-bank to be written to (but not read) in 16k-slot 1, by writing the 16k-bank number to Layer 2 RAM Page Register ($12) and then enabling Layer 2 paging by writing a value with the LSB set to Layer 2 Access Port ($123B).

Writing to this area will then write the appropriate area of memory, whereas reading from it will give the area mapped by other memory management.
