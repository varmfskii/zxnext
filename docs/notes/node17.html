Alternate Page Mode

Timex mode 1

This mode is the same as ZX Spectrum mode at alternate addresses. Alternate page mode is selected by enabling Timex modes by writing 00xxxx1xx to Next register $08 (8, Peripheral 3 setting) then writing 1 to the Timex ULA port ($ff).
