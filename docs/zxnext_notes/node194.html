Miscellaneous calls

m_dosversion $88 (88) get NextZXOS version/mode information        
m_getsetdrv $89 (89) get/set default drive        
m_tapein $8B (8B) tape redirection control (input)        
m_tapeout $8C (8C) tape redirection control (output)        
m_gethandle $8D (8D) get handle for current dot command        
m_getdate $8E (8E) get current date/time        
m_execcmd $8F (8F) execute a dot command        
m_setcaps $91 (91) set additional capabilities        
m_drvapi $92 (92) access API for installable drivers        
m_geterr $93 (93) get or generate error message        
m_p3dos $94 (94) execute +3DOS/IDEDOS/NextZXOS call        
m_errh $95 (95) register dot command error handler
