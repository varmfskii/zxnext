Accessing the zxnDMA

The zxnDMA is mapped to two Read/Write IO Ports 0x0B and 0x6B which are the same ones used by the Datagear but unlike the Datagear, the port at 0x6B runs in zxnDMA mode while the second, port 0x0B, is similar to the MB-02 interface and runs in compatibility mode.

Port $0B (0B) DMA Control (Z80 Mode, 3.01.02)

Port $6B (6B) DMA Control (Next Mode, 3.01.02)
