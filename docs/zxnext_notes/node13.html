ULA Next

The ULA next modes use a varying number of bits from the attribute byte to determine the ink colours as the palette index from the appropriate bits (all others being zero) and the paper colours coming from the indicated value+128 with palette format 255 being a special case where all the bits determine the ink colour while the paper is always palette index 128. The ULA always takes border colour from paper. ULA next is enabled using bit 0 of nextreg $43 (palette control) and controlled with nextreg $42 (ULA Next attribute byte format)
