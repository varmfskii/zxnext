\port{37}{Kempston/Mega Drive Joystick 2}
\begin{itemize}
\item[] bit 7 = ''start'' button
\item[] bit 6 = A/X button
\item[] bit 5 = C/Z button
\item[] bit 4 = Fire/C/Y button
\item[] bit 3 = Up
\item[] bit 2 = Down
\item[] bit 1 = Left
\item[] bit 0 = Right
\end{itemize}
Disable with Nextreg \$05
