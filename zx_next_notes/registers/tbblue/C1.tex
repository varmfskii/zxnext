\register{R/W}{C1}{INT TYP 0} (3.01.08)\\
(\$00 on reset)
\begin{itemize}
\item bits 7-2 = Reserved, must be 0
\item bit 1 = Line
\item bit 0 = ULA
\end{itemize}
* Set bits indicate the corresponding signals are sticky\\
* Reset bits indicate the corresponding signals assert for about 32 CPU cycles
